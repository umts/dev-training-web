    class AppClient

    A wrapper around an Octokit::Client that authenticates as an OAuth application. Contains a few methods for manipulating OAuth tokens as well.

    Public Class Methods

    new(client_id, client_secret) click to toggle source

    Takes two arguments, an OAuth client ID and an OAuth client secret. These are used to initialize an Octokit::Client acting as the OAuth application. This class delegates missing instance methods to that client.

def revoke_token(token)
  return unless token_valid? token

  delete "applications/#{client_id}/grant", access_token: token
end

def token_valid?(token)
  check_application_authorization token, accept: 'application/vnd.github.v3+json'
  true
rescue Octokit::NotFound
  false
end
