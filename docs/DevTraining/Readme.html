    class DevTraining::Readme

    Class representing the “README” file in the training repository. It is responsible for rendering a template file through Tilt and creating the file in the Training repository.

    resource() click to toggle source

    The Sawyer::Resource representing the contents of the readme if found, or the contents and commit information if not found and created with content

    Note that updating a file in a repository is a much more involved process; we don’t attempt it. If any README (in any format) exists in the repository, we let it be.

def resource

end

def repo_relative(path)

  URI.join(base, "#{base.path}/#{path}")
end
